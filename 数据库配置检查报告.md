## 环境变量配置

### 主配置文件: `backend/.env`

配置已按功能分组：

```env
# ================================================
# 数据库配置
# ================================================
DATABASE_URL=
DB_DIALECT=mssql
DB_DRIVER=ODBC Driver 17 for SQL Server
DB_HOST=localhost
DB_PORT=1433
DB_NAME=finflow
DB_USER=admin
DB_PASSWORD=
DB_TRUST_SERVER_CERT=yes
DB_ENCRYPT=yes

# ================================================
# 安全配置
# ================================================
SECRET_KEY=your_secret_key_here
ACCESS_TOKEN_EXPIRE_MINUTES=1440
ENCRYPTION_KEY=
ENCRYPTION_KEY_FILE=

# ================================================
# 马克系统配置
# ================================================
MARKI_USER=your_marki_user
MARKI_PASSWORD=your_marki_password
MARKI_SYSTEM_ID=your_system_id

# ================================================
# 应用配置
# ================================================
APP_HOST=0.0.0.0
APP_PORT=8110
APP_RELOAD=false
ALLOWED_ORIGINS=http://localhost:5273,http://localhost:3000,http://localhost:8110
ALLOW_LAN_ORIGINS=false
```

### 默认值

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `DB_HOST` | `localhost` | 数据库服务器地址 |
| `DB_PORT` | `1433` | SQL Server 默认端口 |
| `DB_NAME` | `finflow` | 数据库名称 |
| `DB_USER` | `admin` | 数据库用户名 |
| `DB_PASSWORD` | (空) | 数据库密码 |
| `DB_DIALECT` | `mssql` | 数据库类型 |
| `DB_DRIVER` | `ODBC Driver 17 for SQL Server` | ODBC 驱动 |

## 结论

✅ **所有主应用代码都正确使用环境变量读取数据库配置**  
✅ **没有硬编码的数据库连接参数**
