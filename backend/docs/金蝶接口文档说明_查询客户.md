# 金蝶云星空接口文档：查询客户

本接口用于在金蝶云星空系统中查询客户基础数据，支持分页和多种条件的组合过滤。

## 1. 接口基本信息

- **用途说明**: 查询客户列表
- **请求方式**: `POST`
- **请求URL**: `/v2/basedata/bd_customer/query`
- **创建者**: administrator

## 2. 请求参数

### 2.1 请求头 (Headers)

| 参数名称 | 必填 | 示例值 | 说明 |
| :--- | :--- | :--- | :--- |
| Content-Type | 是 | `application/json` | 内容格式 |
| accesstoken | 是 | `{{accesstoken}}` | Token 认证方式的请求令牌 |
| Idempotency-Key | 否 | `{{requestId}}` | 幂等性标识，防止接口被重复调用 |
| x-acgw-identity | 是 | `{{identity}}` | 调用 API 的身份标识，第三方应用启用后自动颁发 |

### 2.2 请求体 (Body)

| 参数名称 | 类型 | 必填 | 说明 | 层级 | 示例 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **data** | Object | 否 | 过滤条件对象 | 0 | - |
| ├─ start_createtime | DateTime | 否 | 开始创建时间 | 1 | `"2023-09-01 00:00:00"` |
| ├─ end_createtime | DateTime | 否 | 结束创建时间 | 1 | `"2023-09-05 00:00:00"` |
| ├─ id | Array | 否 | 客户 ID 列表 | 1 | `["310516...", "..."]` |
| ├─ number | Array | 否 | 编码列表 | 1 | `["CUS-0000"]` |
| ├─ createorg_number | Array | 否 | 创建组织编码列表 | 1 | `["BU-0000"]` |
| ├─ status | Array | 否 | 数据状态 [A:暂存, B:已提交, C:已审核] | 1 | `["B","C"]` |
| ├─ idno | Array | 否 | 身份证号列表 | 1 | `["111","222"]` |
| ├─ societycreditcode | Array | 否 | 统一社会信用代码列表 | 1 | `["111","222"]` |
| ├─ bizpartner_phone | Array | 否 | 联系电话列表 | 1 | `["111","222"]` |
| **pageSize** | Integer | 是 | 分页参数：每页数量 | 0 | `10` |
| **pageNo** | Integer | 否 | 分页参数：查询页码 | 0 | `1` |

## 3. 返回参数

| 参数名称 | 类型 | 说明 | 层级 | 示例 |
| :--- | :--- | :--- | :--- | :--- |
| **status** | Boolean | 接口访问是否成功 | 0 | `true` |
| **errorCode** | String | 错误码 (成功为 "0") | 0 | `"0"` |
| **message** | String | 错误信息 | 0 | `""` |
| **data** | Object | 结果数据容器 | 1 | - |
| ├─ filter | String | 操作执行的过滤条件 | 2 | `"billno in (...)"` |
| ├─ lastPage | Boolean | 是否为最后一页 | 2 | `false` |
| ├─ pageNo | Integer | 当前页码 | 2 | `1` |
| ├─ pageSize | Integer | 每页数量 | 2 | `10` |
| ├─ totalCount | String | 总记录数 (不返回时为 "-1") | 2 | `"-1"` |
| └─ **rows** | Array | 客户数据列表 | 2 | - |
| &nbsp;&nbsp;&nbsp;&nbsp;├─ id | Long | 内部 ID | 3 | `"5828580631085078528"` |
| &nbsp;&nbsp;&nbsp;&nbsp;├─ number | String | 客户编码 | 3 | `"CUS-001"` |
| &nbsp;&nbsp;&nbsp;&nbsp;├─ name | String | 客户名称 | 3 | `"客户001"` |
| &nbsp;&nbsp;&nbsp;&nbsp;├─ status | String | 数据状态 | 3 | `"A"` |
| &nbsp;&nbsp;&nbsp;&nbsp;├─ enable | String | 使用状态 ("1" 为启用) | 3 | `"1"` |
| &nbsp;&nbsp;&nbsp;&nbsp;├─ type | String | 伙伴类型 [1:法人, 2:非法人, 3:非企业, 4:个人, 5:个体户] | 3 | `"1"` |
| &nbsp;&nbsp;&nbsp;&nbsp;├─ linkman | String | 联系人 | 3 | `"高先生"` |
| &nbsp;&nbsp;&nbsp;&nbsp;├─ bizpartner_phone | String | 联系电话 | 3 | `"137XXXXXXXX"` |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ ... | ... | 更多字段见 JSON 示例 | 3 | - |
