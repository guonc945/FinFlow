# 会计科目列表查询

## 1. 接口基本信息

- **用途说明**：会计科目列表查询
- **请求方式**：`POST`
- **请求URL**：`/v2/gl/bd_accountview/getList`
- **创建者**：administrator

---

## 2. 请求头参数 (Headers)

| 参数名称 | 参数值 | 说明 |
| :--- | :--- | :--- |
| **Content-Type** | `application/json` | 内容格式 |
| **accesstoken** | 获取的accesstoken值 | Token认证方式的请求令牌 |
| **Idempotency-Key** | 唯一的requestId | 非必传参数，防止接口被重复调用 |
| **x-acgw-identity** | 获取的x-acgw-identity值 | 调用API的身份标识，第三方应用启用后自动颁发 |

---

| 参数名称 | 参数类型 | 必填 | 说明 | 层级 | 示例 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **accounttable_number** | String | 是 | 科目表.编码 | 1 | `"0001"` |
| **accounttype_number** | String | 否 | 会计要素.编码 | 1 | `"0101"` |
| **number** | String | 否 | 编码 | 1 | `"1001"` |
| **name** | String | 否 | 名称 | 1 | `"库存现金"` |
| **isleaf** | Boolean | 否 | 明细科目 | 1 | `false` |
| **longnumber** | String | 否 | 长编码 | 1 | `"1001"` |
| **pageSize** | Integer | 是 | 分页参数，分页数量 | 0 | `10` |
| **pageNo** | Integer | 否 | 分页参数，查询页码 | 0 | `1` |
