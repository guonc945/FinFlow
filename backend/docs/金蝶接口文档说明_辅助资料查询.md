
# 金蝶接口文档说明 - 辅助资料查询

## 1. 接口基本信息

- **用途说明**：查询辅助资料列表详情
- **请求方式**：`POST`
- **请求 URL**：`/v2/base/bos_assistantdata_detail/getList`
- **创建者**：administrator

## 2. 请求头参数 (Headers)

| 参数名称 | 参数值 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| `Content-Type` | `application/json` | 是 | 内容格式 |
| `accesstoken` | 获取的 accesstoken 值 | 是 | Token 认证方式的请求令牌 |
| `Idempotency-Key` | 唯一的 requestId | 否 | 防止接口被重复调用 |
| `x-acgw-identity` | 获取的 x-acgw-identity 值 | 是 | 调用 API 的身份标识，第三方应用启用后自动颁发 |

## 3. 请求体参数 (Body)

| 参数名称 | 参数类型 | 必填 | 层级 | 说明 | 示例 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| `number` | String | 否 | 1 | 编码 | `"8SxZs"` |
| `name` | String | 否 | 1 | 名称 | `"NpGJf"` |
| `ctrlstrategy` | String | 否 | 1 | 控制策略 [7:私有, 5:全局共享] | `"7"` |
| `group_number` | String | 否 | 1 | 类别.编码 | `"9gkpR"` |
| `parent_id` | Long | 否 | 1 | 上级辅助资料.id | `"2545924733561067520"` |
| `parent_number` | String | 否 | 1 | 上级辅助资料.编码 | `"tAxfm"` |
| `createorg_id` | Long | 否 | 1 | 创建组织.id | `"6422829827942609920"` |
| `createorg_number` | String | 否 | 1 | 创建组织.编码 | `"hYpLn"` |
| `pageSize` | Integer | 是 | 0 | 分页参数，每页数量 | `10` |
| `pageNo` | Integer | 否 | 0 | 分页参数，查询页码 | `1` |
