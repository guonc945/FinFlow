# 金蝶接口文档：账簿列表查询

本接口用于查询金蝶系统的账簿列表信息，支持分页、组织名称、核算体系等条件的过滤。

---

## 1. 接口基本信息

- **用途说明**：账簿列表查询
- **请求方式**：`POST`
- **请求 URL**：`/v2/gl/gl_accountbook/getList`
- **维护者**：administrator

---

## 2. 请求头参数 (Headers)

| 参数名称 | 示例值 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| **Content-Type** | `application/json` | 是 | 请求体内容格式 |
| **accesstoken** | `获取的accesstoken值` | 是 | Token 认证令牌 |
| **Idempotency-Key** | `唯一的requestId` | 否 | 防止接口被重复调用 |
| **x-acgw-appid** | `获取的appid` | 否 | 用于网关识别记录 |
| **x-acgw-identity** | `获取的identity` | 否 | 调用 API 的身份标识 |

---

## 3. 请求体参数 (Body)

### 3.1 顶层参数

| 参数名称 | 参数类型 | 必填 | 说明 | 示例 |
| :--- | :--- | :--- | :--- | :--- |
| **pageNo** | `Integer` | 否 | 分页参数：查询页码 | `1` |
| **pageSize** | `Integer` | 是 | 分页参数：每页数量 | `10` |
| **data** | `Object` | 否 | 业务查询条件 | `{}` |

### 3.2 业务数据参数 (data 内部对象)

| 参数名称 | 参数类型 | 必填 | 说明 | 示例 |
| :--- | :--- | :--- | :--- | :--- |
| org_name | `String` | 否 | 核算组织名称 | "胜利产品线" |
| accountingsys_name | `String` | 否 | 核算体系名称 | "责任中心核算体系-02" |
| name | `String` | 否 | 账簿名称 | "金蝶云移动测试副账簿" |
| number | `String` | 否 | 账簿编码 | "00" |
| status | `String` | 否 | 数据状态 (C:已审核 等) | "C" |
| enable | `String` | 否 | 使用状态 (1:正常, 0:禁用) | "1" |

---

## 4. 响应参数说明

### 4.1 顶层响应结构

| 参数名称 | 参数类型 | 说明 |
| :--- | :--- | :--- |
| **status** | `Boolean` | 接口访问是否成功 (true/false) |
| **errorCode** | `String` | 错误码 (成功为 "0") |
| **message** | `String` | 错误信息 (成功为空) |
| **data** | `Object` | 详细业务数据 |

### 4.2 业务数据详情 (data 对象)

| 参数名称 | 类型 | 说明 | 示例 |
| :--- | :--- | :--- | :--- |
| totalCount | `Integer` | 合计记录数 | 1 |
| pageNo | `Integer` | 当前页码 | 1 |
| pageSize | `Integer` | 每页数量 | 10 |
| lastPage | `Boolean` | 是否为最后一页 | true |
| filter | `String` | 执行的过滤条件 | - |
| **rows** | `Array` | 账簿记录列表 (具体见下表) | - |

### 4.3 账簿信息 (rows 内部对象)

| 字段名称 | 类型 | 说明 | 示例 |
| :--- | :--- | :--- | :--- |
| id | `Long` | 唯一标识符 | "4111518229632047104" |
| number | `String` | 编码 | "00" |
| name | `String` | 名称 | "金蝶云移动测试副账簿" |
| org_number | `String` | 核算组织编码 | "000000" |
| org_name | `String` | 核算组织名称 | "胜利产品线" |
| accountingsys_number | `String` | 核算体系编码 | "02" |
| accountingsys_name | `String` | 核算体系名称 | "责任中心核算体系-02" |
| booknature | `String` | 账簿性质 (1:主账簿, 0:副账簿) | "1" |
| accounttable_number | `String` | 科目表编码 | "0002" |
| accounttable_name | `String` | 科目表名称 | "新准则会计科目表" |
| periodtype_name | `String` | 会计日历名称 | "自然月期间" |
| basecurrency_name | `String` | 本位币名称 | "人民币" |
| status | `String` | 状态 (A:暂存, B:已提交, C:已审核) | "A" |
| enable | `String` | 使用状态 | "可用" |

---

## 5. 请求示例 (Payload Sample)

```json
{
    "pageNo": 1,
    "pageSize": 10,
    "data": {
        "org_name": "胜利产品线",

        "name": "金蝶云移动测试副账簿",
        "number": "00",
        "status": "C",
        "enable": "1"
    }
}
```
