# 金蝶接口文档说明

## 1. 凭证新增

- **内容来源**：[https://dev.kingdee.com/open/detail/api/1758916923514425344](https://dev.kingdee.com/open/detail/api/1758916923514425344)

### 接口基本信息

| 属性 | 内容 |
| :--- | :--- |
| **用途说明** | 凭证新增 |
| **请求方式** | POST |
| **请求URL** | `/v2/gl/gl_voucher/voucherAdd` |
| **创建者** | administrator |

### 请求头参数

| 参数名称 | 参数值 | 说明 |
| :--- | :--- | :--- |
| `Content-Type` | `application/json` | 内容格式 |
| `accesstoken` | 获取的accesstoken值 | Token认证方式的请求令牌 |
| `Idempotency-Key` | `1662118878805` | 幂等性参数requestId |
| `x-acgw-identity` | 获取的x-acgw-identity值 | 调用API的身份标识 |

### 请求体参数

| 参数名称 | 参数类型 | 必填 | 说明 | 层级 | 示例 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| `book_number` | String | 是 | 账簿.编码 | 1 | `"BU-35256"` |
| `bizdate` | Date | 是 | 业务日期（业务日期不能晚于记账日期） | 1 | `"2023-11-30"` |
| `bookeddate` | Date | 是 | 记账日期 | 1 | `"2023-11-30"` |
| `period_number` | String | 是 | 期间.编码（会计年度+期间编码） | 1 | `"202311"` |
| `vouchertype_number` | String | 是 | 凭证字.编码 | 1 | `"0001"` |
| `billno` | String | 否 | 凭证号 | 1 | `"4"` |
| `description` | String | 否 | 备注 | 1 | `"测试"` |
| `attachment` | Integer | 否 | 附件数 | 1 | `1` |
| `entries` | Entries | 否 | 分录体 | 1 | `[{},{}]` |
| `- seq` | Integer | 是 | 分录体.分录行号 | 2 | `1` |
| `- edescription` | String | 是 | 分录体.摘要 | 2 | `"摘要"` |
| `- account_number` | String | 是 | 分录体科目.编码 | 2 | `"1005.01"` |
| `- assgrp` | Flex | 否 | 分录体.核算维度 | 2 | `{"客户":{"number":"Cus-504192"}}` |
| `- currency_number` | String | 是 | 分录体币种.货币代码 | 2 | `"CNY"` |
| `- localrate` | Decimal | 是 | 分录体.汇率 | 2 | `1.00` |
| `- debitori` | Decimal | 是 | 分录体.原币借方 | 2 | `3.13` |
| `- creditori` | Decimal | 是 | 分录体.原币贷方 | 2 | `3.13` |
| `- debitlocal` | Decimal | 是 | 分录体.借方 | 2 | `3.13` |
| `- creditlocal` | Decimal | 是 | 分录体.贷方 | 2 | `3.13` |
| `- price` | Decimal | 否 | 分录体.单价 | 2 | `2.5` |
| `- quantity` | Decimal | 否 | 分录体.数量 | 2 | `1.25` |
| `- measureunit` | String | 否 | 计量单位.编码 | 2 | `"09"` |
| `- businessnum` | String | 否 | 分录体.业务编号 | 2 | `"0001"` |
| `- expiredate` | Date | 否 | 分录体.到期日 | 2 | `"2023-11-30"` |
| `- maincfitem` | String | 否 | 主表项目.编码 | 2 | `"CI01.01.01"` |
| `- maincfamount` | Decimal | 否 | 分录体.主表项目金额 | 2 | `-3.13` |
| `- maincfassgrp` | Flex | 否 | 分录体.主表核算维度 | 2 | `{"客户":{"number":"Cus-504192"}}` |
| `- suppcfitem` | String | 否 | 附表项目.编码 | 2 | `"CI05.01.02"` |
| `- suppcfamount` | Decimal | 否 | 分录体.附表项目金额 | 2 | `-3.13` |
| `- settletype_number` | String | 否 | 分录体.结算方式.编码 | 2 | `"JSFS01"` |
| `- settletnumber` | String | 否 | 分录体.结算号 | 2 | `"JSFS22"` |

### 返回参数

| 参数名称 | 参数类型 | 说明 | 层级 | 示例 |
| :--- | :--- | :--- | :--- | :--- |
| `data` | Object | 结果数据 | 1 | `{}` |
| `- result` | Array | 返回结果详细信息 | 2 | `"[]"` |
| `- failCount` | String | 操作失败数量 | 2 | `"2"` |
| `- successCount` | String | 操作成功数量 | 2 | `"1"` |
| `errorCode` | String | 错误码 | 1 | 成功时为0 |
| `message` | String | 接口调用错误信息 | 1 | 成功时为空 |
| `status` | Boolean | 接口访问是否成功 | 1 | `true`/`false` |
