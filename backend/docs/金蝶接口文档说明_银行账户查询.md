接口基本信息
用途说明：
请求方式：POST
请求URL：/v2/basedata/bd_accountbanks/accountbanksQuery
创建者：administrator

请求头参数
参数名称	参数值	说明
Content-Type	application/json	内容格式
accessToken	获取的accessToken值	请求令牌
Idempotency-Key	唯一的requestId	非必传参数，防止接口被重复调用
x-acgw-identity	获取的“x-acgw-identity”值	调用API的身份标识，第三方应用启用后自动颁发
请求体参数
参数名称	参数类型	必填	说明	层级	示例
id	Array	否	内码	1	["5778425353295008768","3874722070946589696"]
bankaccountnumber	Array	否	银行账号	1	["siNYB","K8Ok1"]
name	Array	否	账户简称	1	["deSxP","N63SP"]
acctname	Array	否	账户名称	1	["Jk4wG","sIX5J"]
company_number	Array	否	申请组织.编码	1	["qQubK","TLEOg"]
openorg_number	Array	否	开户组织.编码	1	["ZxGlY","CrAVl"]
status	Array	否	数据状态[A:暂存,B:已提交,C:已审核]	1	["joN1T","wGLgU"]
acctstatus	Array	否	账户状态[normal:正常,closing:销户中,changing:变更中,closed:已销户,freeze:冻结]	1	["Q3oIK","6aymn"]
pageSize	Integer	是	分页参数，分页数量	0	10
pageNo	Integer	否	分页参数，查询页码	0	1
