# 金蝶接口文档 - 查询凭证

## 接口基本信息
- **用途说明**：凭证通过内码查询
- **请求方式**：POST
- **请求URL**：`/v2/gl/gl_voucher/voucherQuery`
- **创建者**：administrator

## 请求头参数

| 参数名称 | 参数值 | 说明 |
| :--- | :--- | :--- |
| Content-Type | application/json | 内容格式 |
| accesstoken | 获取的accesstoken值 | Token认证方式的请求令牌，其它访问策略请查看认证鉴权指南了解。 |
| Idempotency-Key | 唯一的requestId | 非必传参数，防止接口被重复调用 |
| x-acgw-appid | 获取x-acgw-appid的值 | 非必传参数，用于网关识别记录 |
| x-acgw-identity | 获取的x-acgw-identity值 | 调用API的身份标识，第三方应用启用后自动颁发 |

## 请求体参数

| 参数名称 | 参数类型 | 必填 | 说明 | 层级 | 示例 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| id | Long | 是 | 内码ID | 1 | "6188189006934115399" |
| pageSize | Integer | 是 | 分页参数，分页数量 | 0 | 10 |
| pageNo | Integer | 否 | 分页参数，查询页码 | 0 | 1 |
