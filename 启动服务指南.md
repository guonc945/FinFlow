## 🚀 启动服务步骤

### 步骤 1：确保服务已停止
```cmd
sc query FinFlowManagerHost
```
如果显示 `STATE: 4 RUNNING`，则需要停止：
```cmd
sc stop FinFlowManagerHost
```

### 步骤 2：启动 Windows 服务
```cmd
sc start FinFlowManagerHost
```

### 步骤 3：验证服务状态
```cmd
sc query FinFlowManagerHost
```

应该显示：
```
SERVICE_NAME: FinFlowManagerHost
        TYPE               : 10  WIN32_OWN_PROCESS
        STATE              : 4  RUNNING
                                (STOPPABLE, NOT_PAUSABLE, IGNORES_SHUTDOWN)
```

### 步骤 4：检查进程
```cmd
tasklist | findstr FinFlow
tasklist | findstr python
tasklist | findstr node
```

### 步骤 5：检查运行时状态
```cmd
type deploy\windows\runtime\service_runtime.json
```

应该显示：
```json
{
  "service_host_pid": <PID>,
  "backend_pid": <PID>,
  "frontend_pid": <PID>,
  "host_last_transition": "running",
  ...
}
```

### 步骤 6：查看服务日志
```cmd
type backend\logs\manager.service.log
```

## 🔧 如果服务启动失败

### 检查端口占用
```cmd
netstat -ano | findstr 8110
netstat -ano | findstr 5273
```

如果有占用，停止相关进程：
```cmd
taskkill /F /PID <进程ID>
```

### 检查配置文件
确保 `deploy/windows/runtime/manager_state.json` 中包含：
```json
{
  "start_backend_on_launch": true,
  "start_frontend_on_launch": true
}
```

### 完整重启流程
```cmd
# 1. 停止服务
sc stop FinFlowManagerHost

# 2. 清理进程
taskkill /F /IM FinFlowManager.exe 2>nul
taskkill /F /IM FinFlowManagerHost.exe 2>nul
taskkill /F /IM python.exe 2>nul
taskkill /F /IM node.exe 2>nul

# 3. 等待 3 秒
timeout /t 3

# 4. 启动服务
sc start FinFlowManagerHost

# 5. 检查状态
sc query FinFlowManagerHost
```
