# FinFlow SQL Server 配置修正说明

## 修正内容

### 问题描述

之前数据库配置逻辑有误：
- **错误**: `database.txt` 文件的优先级高于 `.env` 环境变量
- **正确**: `.env` 环境变量应该具有最高优先级，`database.txt` 仅作为备注

### 修正的文件

#### 1. `backend/database.py`

**修改点**:

1. **移除 `_prefer_file_values` 变量**
   ```python
   # 之前
   _prefer_file_values = DB_DIALECT in {"mssql", "sqlserver", "sql_server"}
   
   # 修正后
   # (已移除)
   ```

2. **修改 `_db_setting()` 函数逻辑**
   ```python
   # 之前：优先使用文件值
   def _db_setting(name: str, default: str) -> str:
       if _prefer_file_values:
           file_value = _file_db_config.get(name, "").strip()
           if file_value:
               return file_value
       
       env_value = os.getenv(name, "").strip()
       if env_value:
           return env_value
       
       file_value = _file_db_config.get(name, "").strip()
       if file_value:
           return file_value
       
       return default
   
   # 修正后：优先使用环境变量
   def _db_setting(name: str, default: str) -> str:
       # 优先使用环境变量，其次才是文件值
       env_value = os.getenv(name, "").strip()
       if env_value:
           return env_value
       
       file_value = _file_db_config.get(name, "").strip()
       if file_value:
           return file_value
       
       return default
   ```

3. **移除 `database.txt` 中的 `数据库类型` 解析**
   ```python
   # 之前
   if "type" in key or "dialect" in key:
       config["DB_DIALECT"] = value
   elif "host" in key or "server" in key:
       config["DB_HOST"] = value
   
   # 修正后
   # database.txt 只作为备注使用，不解析数据库类型
   if "host" in key or "server" in key:
       config["DB_HOST"] = value
   ```

4. **简化默认值**
   ```python
   # 之前
   DB_PORT = _db_setting("DB_PORT", "1433" if _prefer_file_values else "5432")
   DB_USER = _db_setting("DB_USER", "admin" if _prefer_file_values else "postgres")
   
   # 修正后
   DB_PORT = _db_setting("DB_PORT", "1433")
   DB_USER = _db_setting("DB_USER", "admin")
   ```

## 配置优先级

### 读取顺序

1. **第一优先**: 环境变量 (`.env` 文件)
2. **第二优先**: `database.txt` 文件 (仅作备注)

### 示例

如果 `.env` 和 `database.txt` 配置不同：
```
.env:           DB_HOST=localhost
database.txt:   服务器：192.168.1.100
```

**实际使用**: `localhost` (来自 `.env`)

## 配置文件说明

### `.env` 文件 (主配置)

**位置**: `backend/.env`

**用途**: 所有数据库连接参数

**示例**:
```env
DB_DIALECT=mssql
DB_DRIVER=ODBC Driver 17 for SQL Server
DB_HOST=localhost
DB_PORT=1433
DB_NAME=finflow
DB_USER=admin
DB_PASSWORD=your_password
DB_TRUST_SERVER_CERT=yes
DB_ENCRYPT=yes
```

### `database.txt` 文件 (备注)

**位置**: `database.txt`

**用途**: 仅作为配置参考，不参与实际解析

**示例**:
```text
数据库配置备注 (此文件仅作参考，实际配置请修改 backend/.env)

数据库类型：sqlserver
服务器：fn.hyqy.group
端口：1433
账号：sa
密码：sa@sqlserver
```

## 使用方法

### 修改数据库配置

1. 编辑 `backend/.env` 文件
2. 修改对应的配置项
3. 重启后端服务

**示例**:
```env
# 修改数据库服务器
DB_HOST=192.168.1.100

# 修改数据库端口
DB_PORT=1433

# 修改数据库名称
DB_NAME=finflow_prod

# 修改数据库用户
DB_USER=sa

# 修改数据库密码
DB_PASSWORD=your_password
```

### 验证配置

```bash
# 检查 .env 文件
type backend\.env

# 检查 database.txt (仅备注)
type database.txt
```
