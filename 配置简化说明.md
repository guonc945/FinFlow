### 主要变更

1. **移除所有对 `database.txt` 的引用**
   - 移除 `DB_CONFIG_FILE` 变量
   - 移除 `DB_USE_CONFIG_FILE` 变量
   - 移除 `_parse_db_config_file()` 函数
   - 移除 `_file_db_config` 变量
   - 移除 `_db_setting()` 函数

2. **简化配置读取**
   - 直接从环境变量读取配置
   - 移除复杂的文件解析逻辑
   - 移除不必要的默认值逻辑

3. **代码行数减少**
   - 从 ~195 行减少到 ~101 行
   - 减少了 ~94 行代码

## 配置文件

### `.env` 文件 (主配置)

**位置**: `backend/.env`

**用途**: 所有数据库连接参数

**示例**:
```env
DB_DIALECT=mssql
DB_DRIVER=ODBC Driver 17 for SQL Server
DB_HOST=localhost
DB_PORT=1433
DB_NAME=finflow
DB_USER=admin
DB_PASSWORD=your_password
DB_TRUST_SERVER_CERT=yes
DB_ENCRYPT=yes
```

### `database.txt` 文件 (备注)

**位置**: `database.txt`

**用途**: 仅作为配置参考，不参与实际解析

**示例**:
```text
数据库配置备注 (此文件仅作参考，实际配置请修改 backend/.env)

数据库类型：sqlserver
服务器：fn.hyqy.group
端口：1433
账号：sa
密码：sa@sqlserver
```

## 配置优先级

### 读取顺序

**只有一种**: `.env` 环境变量

`database.txt` 不参与配置解析，仅作为备注。

## 使用方法

### 修改数据库配置

1. 编辑 `backend/.env` 文件
2. 修改对应的配置项
3. 重启后端服务

**示例**:
```env
# 修改数据库服务器
DB_HOST=192.168.1.100

# 修改数据库端口
DB_PORT=1433

# 修改数据库名称
DB_NAME=finflow_prod

# 修改数据库用户
DB_USER=sa

# 修改数据库密码
DB_PASSWORD=your_password
```

### 验证配置

```bash
# 检查 .env 文件
type backend\.env
```
